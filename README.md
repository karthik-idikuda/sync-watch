# SyncWatch - Virtual Theater

Watch videos together with friends in real-time! 🎬

## Features
- 🎥 Multi-platform support (YouTube, Vimeo, Twitch, HLS streams, direct URLs)
- 🔄 Real-time video synchronization
- 💬 Live chat with emoji reactions
- 📹 Video/Audio chat via WebRTC
- 🎨 Futuristic cyberpunk UI
- 📱 Responsive design

## Quick Start
```bash
npm install
npm start
```

Then open http://localhost:3000

## Tech Stack
- Node.js + Express
- Socket.io
- WebRTC
- HLS.js
